# 🌟 About Abhyasa

**Abhyasa** is a minimalist habit tracker designed to help you build meaningful routines through consistency, reflection, and mindfulness.

Most habit trackers are packed with social feeds, complicated challenges, intrusive ads, and endless notifications.

Abhyasa takes a different approach.

It focuses on the fundamentals:

- Building habits consistently
- Tracking meaningful progress
- Staying mindful
- Maintaining privacy
- Keeping distractions away

Whether you're trying to meditate daily, drink more water, exercise regularly, or simply become more intentional with your time, Abhyasa helps you stay accountable to yourself.

---

# 📖 Our Story

Abhyasa started as a personal project — a simple way to track daily habits without the noise of social features, gamification overload, or subscription paywalls.

The developer behind it kept forgetting to drink water, skipping meditation, and falling off exercise routines.

Every habit tracker felt either:

- Too complex
- Too distracting
- Too childish
- Too expensive

So something different was built.

A privacy-first habit tracker that treats users like adults.

No streaks lost because your internet was unavailable.

No endless social notifications.

No ads trying to sell productivity supplements.

No premium subscription required for basic functionality.

Just honest habit tracking designed to help people improve one day at a time.

# 🧘 Philosophy

The word **Abhyasa (अभ्यास)** comes from Sanskrit.

It means:

> Consistent practice performed with dedication over a long period of time.

Real change doesn't happen through motivation.

Real change happens through repetition.

One small action.

Repeated daily.

That's the foundation of Abhyasa.

# 📦 Installation

## Clone Repository

```bash
git clone https://github.com/yourusername/abhyasa.git
```

```bash
cd abhyasa
```

## Install Dependencies

```bash
npm install
```

or

```bash
yarn install
```

## Start Development Server

```bash
npx expo start
```

---

# 🔥 Firebase Setup

Create a Firebase project and configure:

- Authentication
- Firestore
- Cloud Messaging

Add Firebase configuration:

```env
EXPO_PUBLIC_FIREBASE_API_KEY=
EXPO_PUBLIC_FIREBASE_AUTH_DOMAIN=
EXPO_PUBLIC_FIREBASE_PROJECT_ID=
EXPO_PUBLIC_FIREBASE_STORAGE_BUCKET=
EXPO_PUBLIC_FIREBASE_MESSAGING_SENDER_ID=
EXPO_PUBLIC_FIREBASE_APP_ID=
```
